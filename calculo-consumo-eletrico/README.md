## 📌 Sobre o projeto

Este é um programa simples feito em Python para calcular o consumo mensal de energia elétrica de um aparelho em kwh.

O usuário informa o nome do aparelho, a potência e o tempo de uso diário, e o sistema calcula o consumo mensal.

---

## 🧠 Fórmula utilizada

consumo_mensal = (potencia * horas_dia * 30) / 1000
