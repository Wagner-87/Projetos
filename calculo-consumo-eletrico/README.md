## 📌 Sobre o projeto

Este é um programa simples feito em Python para calcular o consumo e custo mensal de energia elétrica de um aparelho.

O usuário informa o nome do aparelho, a potência, o tempo de uso diário e o preço do kWh. O sistema então calcula o consumo e o custo mensal.

---

## 🎯 Funcionalidades

- Inserir nome do aparelho
- Informar potência (em watts)
- Informar tempo de uso diário
- Informar preço do kWh
- Calcular consumo mensal
- Calcular custo mensal

---

## 🧠 Fórmulas utilizadas

```
consumo_mensal = (potencia * horas_dia * 30) / 1000
custo_mensal = consumo_mensal * preco_kwh
```
