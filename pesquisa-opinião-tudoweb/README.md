## 📌 Sobre o projeto

Este projeto foi desenvolvido como atividade acadêmica para praticar conceitos básicos de programação em Python.

O programa realiza uma **pesquisa de satisfação com clientes**, coletando informações e exibindo resultados ao final.

---

## 🎯 Objetivo

Criar um sistema simples que:

- Solicita o **nome** do cliente
- Solicita a **idade**
- Coleta a **opinião sobre o atendimento**

---

## 📝 Opções de resposta

| Código | Opinião     |
|--------|------------|
| 1      | EXCELENTE  |
| 2      | BOM        |
| 3      | RUIM       |

---

## ⚙️ Funcionamento

- O programa entrevista **50 pessoas**
- Utiliza um laço de repetição `for`
- Usa estrutura condicional `if/elif`
- Conta quantas respostas foram:
  - ✅ EXCELENTE
  - ❌ RUIM
- Exibe o resultado final na tela

---

1. Instale o Python (3.x)
2. Baixe ou clone este repositório
3. Abra o terminal na pasta do projeto
4. Execute o comando:
